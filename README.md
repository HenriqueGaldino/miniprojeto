# Controle de Hábitos

## 👤 Nome
Henrique 

## 📱 Descrição
Aplicativo desenvolvido em Flutter para controle de hábitos diários.

## ⚙️ Funcionalidades
- Lista de hábitos
- Marcar como concluído
- Tela de detalhes
- Navegação entre telas
- Simulação de carregamento

## ▶️ Como rodar
flutter pub get
flutter run
